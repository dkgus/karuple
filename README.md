# 🏁 카러플 MBTI 테스트

카트라이더 캐릭터를 16가지 유형으로 분류하여 진행해 볼 수 있는 MBTI 테스트 웹 앱입니다.

---

## 🎯 프로젝트 목표 및 배경

- **카트라이더 캐릭터**의 성격과 특징을 기반으로 16가지 MBTI 유형을 구성하였습니다.
- 간단한 테스트를 통해 사용자의 성향과 어울리는 캐릭터를 추천합니다.
- 누구나 쉽게 즐길 수 있는 **반응형 웹 서비스**를 목표로 제작하였습니다.
